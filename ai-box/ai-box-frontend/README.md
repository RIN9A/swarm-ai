# AI Box - Конструктор цифровых сотрудников

Современный веб-интерфейс для создания и управления ИИ-агентами для бизнеса.

## Возможности

- Создание агентов из готовых шаблонов (Юрист, Бухгалтер, Маркетолог, Дизайнер)
- Пошаговый wizard для настройки агентов (4 шага)
- Выбор инструментов для агентов (PDF парсер, Email, Таблицы, 1C API и др.)
- Настройка параметров языковой модели (модель, температура, итерации)
- Запуск агентов и просмотр результатов
- Просмотр логов выполнения
- Статистика по запускам и успешности

## Технологии

- **Next.js 13** (App Router)
- **React 18**
- **TypeScript**
- **Tailwind CSS**
- **Supabase** (база данных)
- **Lucide React** (иконки)

## Установка и запуск

### 1. Установите зависимости

```bash
npm install
```

### 2. Настройка переменных окружения

Файл `.env` уже настроен с подключением к Supabase.

### 3. Запустите проект в режиме разработки

```bash
npm run dev
```

Приложение откроется на `http://localhost:3000`

### 4. Сборка для продакшена

```bash
npm run build
npm start
```

```
app/

  Header.tsx                  # Шапка приложения
  AgentCard.tsx               # Карточка агента
  AgentDetails.tsx            # Детали агента
  EmptyState.tsx              # Пустое состояние
  ExecutionPanel.tsx          # Панель запуска агента
  wizard/
    CreateAgentWizard.tsx     # Главный компонент wizard
    StepIndicator.tsx         # Индикатор шагов
    Step1Role.tsx             # Шаг 1: Выбор роли
    Step2Info.tsx             # Шаг 2: Основная информация
```

## База данных

В Supabase созданы следующие таблицы:

- `agents` - хранение информации об агентах
- `agent_executions` - история запусков агентов

Демо-данные уже загружены в базу (3 примера агентов).

## Экраны приложения

### 1. Главная страница
- Список всех агентов в виде карточек
- Кнопка создания нового агента
- Пустое состояние, если агентов нет

### 2. Мастер создания агента
- **Шаг 1**: Выбор роли из шаблонов
- **Шаг 2**: Заполнение имени, описания и system prompt
- **Шаг 3**: Выбор инструментов (чекбоксы)
- **Шаг 4**: Настройка параметров модели + предпросмотр JSON

### 3. Страница агента
- Детали агента (имя, роль, описание, инструменты)
- Статистика (запуски, успешность)
- Панель запуска с полем ввода
- Вывод результатов и логов выполнения

## Особенности дизайна

- Спокойная цветовая палитра (синий, бирюзовый, серый)
- Градиенты для акцентных элементов
- Закругленные карточки с тенями
- Адаптивная верстка (desktop first)
- Плавные анимации и переходы
- Понятные иконки и текста на русском языке
